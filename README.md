## 📌 Project Description

The **Weather Prediction App** is a web-based application that fetches and displays real-time weather data for any searched city.  
Users can instantly view key weather details such as:

- 🌡️ Temperature  
- 💧 Humidity  
- 🌬️ Wind Speed  
- ☁️ Weather Condition

The app is designed to be simple, fast, and beginner-friendly while showcasing solid frontend and API-handling skills.

## 🌐 API Used

This project uses the **OpenWeatherMap API** for weather and geolocation data.

- Current Weather API
- 5-Day / 3-Hour Forecast API
- Reverse Geocoding API

Official API Docs: [https://openweathermap.org/api](https://openweathermap.org/api)

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/BolugarlaSathvick10/Weatherapp.git

# 2. Move into the project folder
cd Weatherapp

# 3. Install dependencies
npm install

# 4. Start development server
npm run dev
```

To serve the project as a static app:

```bash
npm start
```

---

## 📖 Usage

1. Open the app in your browser.  
2. Enter a city name in the search bar.  
3. View weather details like temperature, humidity, wind speed, and conditions.  
4. Explore the responsive layout on different screen sizes.
